PersCraft – Static Website (HTML/CSS)

Start:
- Öffne index.html im Browser, oder hoste den Ordner bei einem statischen Hoster.

Hinweise:
- In impressum.html bitte Pflichtangaben ergänzen (Adresse, Telefon, USt-ID etc.).
- In datenschutz.html ggf. Hosting/Tools ergänzen (Analytics, Maps, Fonts, Cookie-Banner).

Struktur:
- /assets/css/style.css
- /assets/img/ (Bilder)
